Inbuilt PDF Viewer

After installing this module, you will able to open PDF reports inside OpenERP and you can also able to see reports if multiple reports printed from tree view.

Install appropriate plugins if reports are not displaying properly.

(Required for old version browsers only)


  • For Google Chrome or Chromium

    • Click to Download: Docs PDF/PowerPoint Viewer (by Google)


    • Click to Download: PDF.js


  • For Mozilla Firefox

    • Click to Download: PDF.js for Mozilla Firefox
